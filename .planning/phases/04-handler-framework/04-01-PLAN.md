    - `private val scope = CoroutineScope(Dispatchers.IO + SupervisorJob() + CoroutineExceptionHandler { _, e -> logger.error(e) { "..." } })` (D-02, D-04)
    - `suspend fun dispatch(envelopeRequest: Request): Response` (D-15):
