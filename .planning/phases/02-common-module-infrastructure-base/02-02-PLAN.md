    在 `common/src/main/kotlin/com/nebula/common/idgen/SnowflakeIdGenerator.kt` 实现雪花算法生成器（per 设计文档 9.4 + D-13/D-14）：

    位分配：41 位时间戳（毫秒）| 10 位 Worker ID | 12 位序列号

    1. **类构造器参数**：`workerId: Long`、`epoch: Long = 1700000000000L`
    2. **常量**：workerIdBits=10L、sequenceBits=12L、maxWorkerId=1023L、sequenceMask=4095L、workerIdShift=12、timestampShift=12+10=22
    3. **可变状态**：`lastTimestamp: Long = -1L`、`sequence: Long = 0L`
    4. **init 块**：`require(workerId in 0..maxWorkerId)` 验证 Worker ID 范围
    5. **`@Synchronized fun nextId(): Long`**:
       - 获取当前毫秒时间戳
       - 如果 `timestamp < lastTimestamp`：计算差值 diff，`throw ClockBackwardsException("Clock moved backwards $diff ms...")`（per D-13）
       - 如果 `timestamp == lastTimestamp`：sequence++ 并 mask，如果 sequence 溢出为 0 则调用 `waitNextMillis()`（per D-14 自愈）
       - 否则（timestamp > lastTimestamp）：sequence 归零
       - 写入 `lastTimestamp = timestamp`
       - 返回 `((timestamp - epoch) shl timestampShift) or (workerId shl workerIdShift) or sequence`
    6. **`@Synchronized private fun waitNextMillis(lastTimestamp: Long): Long`**：忙等直到 `System.currentTimeMillis() > lastTimestamp`
  </action>
