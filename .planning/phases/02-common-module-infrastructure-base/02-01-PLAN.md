       - out → `config/dev/ssl/fullchain.pem`
       - 执行 `mkdir -p config/dev/ssl`
       - `-subj "/CN=localhost"` 
       - `-addext "subjectAltName=DNS:localhost,IP:127.0.0.1"`

       - 脚本开头写 `#!/bin/bash`

       - pattern: `%d{HH:mm:ss.SSS} %highlight(%-5level) [%thread] %cyan(%logger{36}) - %msg%n`

    3. 创建 `common/src/main/resources/logback-prod.xml`（per D-17）：
